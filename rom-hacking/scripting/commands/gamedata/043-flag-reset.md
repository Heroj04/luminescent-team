# (43) _FLAG_RESET

## Effect

Resets the given flag to false.

## Syntax

```c
_FLAG_RESET(flag)
```

| Argument | Description | Types | Required |
| - | - | - | - |
| **flag** | The flag to reset | Flag | Required |

## Example

```c
_FLAG_RESET(#FH_03)
```

The above script will reset the #FH_03 flag.
