# (783) _AC_MARK_GYOE

## Effect

Displays an exclamation mark bubble \[!\] above the entity's head.

The bubble lasts for 1.5 seconds.

## Syntax

```c
_AC_MARK_GYOE()
```

## Example

```c
anm_dummy:
_AC_MARK_GYOE()
_ACMD_END()
```

The above script will display an exclamation mark \[!\] above the entity's head.
